# Input to Chemical Symbols

This repository aims to take in an input from user and turns the input into a string of chemical names based on its short form.

Examples: <br>
(1) <br>
Input: boy <br>
Output: <br>
B: Boron <br>
O: Oxygen <br>
Y: Yttrium <br>

(2) 
Input: win <br>
Output: <br>
W: Tungsten <br>
In: Indium <br>
